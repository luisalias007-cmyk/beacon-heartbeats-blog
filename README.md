# Getting Started with Beacon 2.x: Heartbeats for AI Agents

This repository publishes a small personal-blog article for the RustChain Beacon tutorial bounty:

https://luisalias007-cmyk.github.io/beacon-heartbeats-blog/

It includes:

- `index.html`: the tutorial article.
- `contracts-mayday.html`: a second tutorial focused on contracts and mayday escalation.
- `rss.xml`: RSS feed for discoverability.

The articles link back to the official beacon-skill repository and include tested self-contained Python examples for signed heartbeats and mayday escalation.
